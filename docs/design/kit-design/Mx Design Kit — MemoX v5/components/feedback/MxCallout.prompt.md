Inline context banner — info, warning, destructive, no-due, unavailable. Icon + colour + text.

```jsx
<MxCallout tone="noDue" title="Nothing due">You’ve reviewed everything today.</MxCallout>
<MxCallout tone="warning">Changing the app icon may remove home-screen shortcuts.</MxCallout>
```

`noDue` is calm/positive — never styled as an error.
