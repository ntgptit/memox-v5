Mx Kit QA Checklist

Run this before any MemoX product screen is designed on top of the kit. It validates that the kit is complete enough to design and implement from — naming, dark/light parity, component coverage, composition, states, and React Native handoff. Boxes are for a single review pass (state is not saved).

Quality bar The kit passes only when a future designer can build screens using only these pages, an RN agent can implement the shared components from them, and both themes plus all non-happy states are documented and unambiguous.
